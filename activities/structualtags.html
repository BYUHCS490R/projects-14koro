Structual Tags

header

The header tag is used for introductory content such as site title.

nav

The navigation tag is used for sets of navigational links. It can be in the header, but it does not have to be.

main

The main tag holds main content of that page.

footer

The footer tag is used for informations at the bottom of the page.

Questions

What is the difference between block tags and inline tags?

Block tags refer to elements like the `main` tag or `p` tag, encompassing an entire line. Inline tags refer to elements like the `span` tag or `a` tag, containing only the specified portion of the element.

What is the margin, border, and padding?

margin refers to the space outside an element, padding refers to the space inside an element, and border refers to the line that outlines an element.

What are units of measurement you can use in CSS?

  • px
  • %
  • in
  • cm

When would you use a figure tag?

The figure tag can be used to provide meaning to a figure and its associated description as a set.

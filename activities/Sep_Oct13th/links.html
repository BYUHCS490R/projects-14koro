Questions

2. What is the root? What is the relationship between folder and file names and the path?

The root is the top-level folder of your website. A path shows the directions from the root folder to the file.

3. What is a default document?

A default document is a file that is automatically loaded when you access a directory.

4. Explain the difference between an absolute link and a relative link. Why would you use relative links?

Absolute links are used when referencing other websites. In such cases, use the full address. Relative links are used when referencing within files you have created. Using relative links is easier.

6. List the 3 rules with relative links.

  1. Inside a folder
  2. Up and then down a folder
  3. Start over

8. Do links always need to be inside a NAV tag?

No. Nav is used for a set of main navigation links, but links can appear anywhere.

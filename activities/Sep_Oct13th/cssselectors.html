What is the difference between an ID and a Class?

An ID is used for a specific single instance, while a class can be reused repeatedly for multiple instances.

What is the difference between a SPAN tag and a DIV tag?

Use SPAN tags for specific text elements, and DIV tags for larger sections.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

The div tag has no specific meaning. You can use it only when nothing else fits.

Questions

Explain the difference between inline, embedded, and external CSS.

Inline CSS is a method of modifying styles by writing them directly into tags. Embedded CSS is a method of modifying styles for the entire HTML document by placing a style tag within the head section. External CSS is a method of modifying styles by placing information in an external link file and applying changes via a link.

Explain why you should only use external CSS.

Using external CSS makes loading faster and keeps HTML simple and easy to read.

What is a selector used for?

A selector is the part that specifies “where to apply” when using CSS.
